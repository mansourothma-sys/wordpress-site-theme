# Sirte E-Learning Center — WordPress Theme (v2.0.0)

قالب WordPress عربي RTL لمركز التعليم الإلكتروني بجامعة سرت. النسخة 2.0.0 تحوّل الموقع من صفحة واحدة إلى موقع متعدد الصفحات: الرئيسية، عن المركز، الكليات والمقررات، الحوكمة والوثائق، الأخبار والفعاليات، أدلة الاستخدام، اتصل بنا — مع نموذج تواصل فعلي، صفحة 404 مخصصة، وتحسينات بصرية (عدّاد أرقام، خط ثانٍ Tajawal للنصوص، أكورديون أسئلة شائعة، تايم لاين للمسيرة المؤسسية).

## ما الجديد في هذه النسخة

### صفحات جديدة (Page Templates)
كل ملف أدناه هو "قالب صفحة" جاهز — يظهر في WordPress عند إنشاء صفحة جديدة ضمن "الصفحة الأصل > القالب":

| الملف | اسم القالب | الرابط المقترح (slug) |
|---|---|---|
| `page-about.php` | صفحة عن المركز | `about` |
| `page-academics.php` | صفحة الكليات والمقررات | `academics` |
| `page-governance.php` | صفحة الحوكمة والوثائق | `governance` |
| `page-news.php` | صفحة الأخبار والفعاليات | `news` |
| `page-guides.php` | صفحة أدلة الاستخدام | `guides` |
| `page-contact.php` | صفحة اتصل بنا | `contact` |
| `page.php` | (تلقائي) | أي صفحة أخرى بدون قالب مخصص |
| `404.php` | (تلقائي) | صفحة الخطأ 404 |

القائمة الرئيسية والتذييل يتوليان الربط تلقائيًا بهذه الصفحات عبر الدالة `sirte_elc_page_url()` في `inc/theme-data.php`، بشرط أن يكون الـ slug مطابقًا للجدول أعلاه.

### أقسام جديدة قابلة لإعادة الاستخدام (`template-parts/sections/`)
- `timeline.php` — مسيرة تأسيس المركز (بيانات من `sirte_elc_history()`).
- `org-units.php` — الوحدات التنظيمية (بيانات من `sirte_elc_org_units()`).
- `guides.php` — شبكة أدلة الاستخدام حسب الفئة (طلاب / أعضاء هيئة تدريس).
- `faq.php` — أسئلة شائعة قابلة للطي (بيانات من `sirte_elc_faq()`).
- `counters.php` — شريط أرقام حقيقية (عدد الكليات، عدد الوثائق...) وليست أرقامًا تسويقية مُتخيَّلة.
- `cta-band.php` — بانر دعوة لإجراء قابل لإعادة الاستخدام في أي صفحة.

### نموذج اتصل بنا يعمل فعليًا
`page-contact.php` يرسل عبر `wp_mail()` إلى `elcsupport@su.edu.ly` من خلال معالج آمن في `functions.php`:
- تحقق CSRF عبر `wp_nonce_field` / `wp_verify_nonce`.
- حقل Honeypot مخفي لصد الرسائل الآلية.
- إعادة توجيه بعد الإرسال مع رسالة نجاح/فشل واضحة.

> ملاحظة: نجاح الإرسال يعتمد على أن تكون استضافة الموقع مهيأة لإرسال بريد (SMTP)، غالبًا عبر إضافة مثل WP Mail SMTP. إن لم تُرسل الرسائل، هذا أول ما يجب فحصه.

### الأخبار والفعاليات بدون محتوى مُختلَق
`page-news.php` تسحب مقالات WordPress الحقيقية (مقالات > أضف جديد) عبر `WP_Query` — لا يوجد أي خبر أو تاريخ مُلفَّق؛ إن لم تُنشر أي مقالة بعد تظهر رسالة "لا توجد أخبار منشورة بعد" بدل بيانات وهمية.

### أرقام وإحصاءات حقيقية فقط
شريط العدادات في `counters.php` يستخدم:
- عدد الكليات الفعلي من `sirte_elc_faculties()`.
- عدد وثائق الحوكمة الفعلي من `sirte_elc_documents()`.
- عدد مجموعات الأدلة الفعلي من `sirte_elc_guides()`.

لم تُضَف أي أرقام تسويقية مثل "عدد الطلاب" أو "سنوات الخبرة" لأنها غير مؤكدة — إن رغبت بإضافتها زوّدني بالرقم الدقيق وسأدرجه.

## خطوات التركيب والتفعيل (لكودكس / منصور)

1. انسخ محتوى هذا المستودع إلى: `wp-content/themes/sirte-elc`
2. من لوحة التحكم: **المظهر > قوالب** → فعّل قالب **Sirte E-Learning Center**.
3. **الإعدادات > الروابط الدائمة** تأكد أنها ليست "عادي" (Plain) حتى تعمل الروابط `/about/` وغيرها بشكل صحيح.
4. أنشئ 6 صفحات جديدة (**صفحات > أضف جديد**) بنفس الأسماء والـ slugs في الجدول أعلاه، وفي كل صفحة من قائمة "القالب" على اليمين اختر القالب المطابق.
5. **المظهر > القوائم**: أنشئ قائمة جديدة، أضف الصفحات الست + الرئيسية، وعيّنها لموقع "القائمة الرئيسية" (`primary`). إن لم تُنشئ قائمة يدويًا فسيظهر القالب الافتراضي المُضمَّن تلقائيًا (نفس الروابط بالضبط).
6. لو أردت تفعيل نموذج اتصل بنا فعليًا، ثبّت إضافة SMTP (مثل WP Mail SMTP) واربطها بحساب بريد `elcsupport@su.edu.ly` أو أي بريد آخر تفضّله.
7. صورة الـ Hero الحالية مأخوذة من منصة Moodle القديمة (`template-parts/sections/hero.php`، متغيّر `$hero_image`) — **لا تزال المهمة المعلّقة**: تزويدي بصورة حرم جامعي عالية الجودة لاستبدالها.

## تعديل بيانات الكليات

افتح `inc/theme-data.php` → عدّل عناصر `sirte_elc_faculties()`. كل كلية: `name`, `logo`, `url`.

## تعديل وثائق الحوكمة

افتح `inc/theme-data.php` → عدّل عناصر `sirte_elc_documents()`. كل وثيقة: `title`, `description`, `url`, `icon` (المتاح: `network`, `shield`, `target`, `book`).

## تعديل المسيرة المؤسسية / الوحدات التنظيمية / الأسئلة الشائعة / الأدلة

كلها في `inc/theme-data.php` ضمن الدوال: `sirte_elc_history()`, `sirte_elc_org_units()`, `sirte_elc_faq()`, `sirte_elc_guides()`. أضف/عدّل مصفوفة PHP بنفس البنية الظاهرة.

```

page-about.php          عن المركز
page-academics.php      الكليات والمقررات
page-governance.php     الحوكمة والوثائق
page-news.php           الأخبار والفعاليات (مقالات WordPress حقيقية)
page-guides.php         أدلة الاستخدام
page-contact.php        اتصل بنا (نموذج فعلي)
page.php                قالب عام لأي صفحة أخرى
404.php                 صفحة الخطأ المخصصة
header.php / footer.php الرأس والتذييل (روابط تلقائية لكل الصفحات)
functions.php           إعدادات القالب + معالج نموذج التواصل
inc/theme-data.php       كل المحتوى النصي والبيانات القابلة للتعديل
inc/template-tags.php    دوال عرض مساعدة (أيقونات، عناوين أقسام، هيدر الصفحة الداخلية)
template-parts/sections/ كل قسم من أقسام الصفحات كملف مستقل
assets/css/main.css            التنسيق الأساسي (من النسخة السابقة)
assets/css/visual-upgrade.css  تحسينات بصرية إضافية (من النسخة السابقة)
assets/css/pages.css           تنسيق الصفحات الداخلية الجديدة (v2.0.0)
assets/js/main.js              قائمة الجوال + فلترة الكليات
assets/js/scroll-reveal.js     حركة الظهور عند التمرير
assets/js/interactions.js      أكورديون الأسئلة الشائعة + عدّاد الأرقام + حالة "لا نتائج" في البحث
```

## الملاحظات العامة

- القالب يعتمد على خط Cairo للعناوين وخط Tajawal للنصوص، كلاهما من Google Fonts.
- لا توجد حاجة إلى إضافات خارجية لتشغيل الصفحات الأساسية؛ نموذج التواصل فقط يحتاج SMTP كما ذُكر أعلاه.
- جميع الأرقام والإحصاءات المعروضة مُشتقّة من بيانات حقيقية في `theme-data.php`، ولا يوجد أي محتوى وهمي (أسماء أشخاص، شهادات مستخدمين، أو تواريخ غير مؤكدة).
