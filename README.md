# Sirte E-Learning Center WordPress Theme

قالب WordPress عربي RTL لمركز التعليم الإلكتروني بجامعة سرت. القالب يبني بوابة رسمية للمركز تربط الزائر بمنصة Moodle، وتعرض الرؤية والرسالة، بوابة كليات الجامعة، وثائق الحوكمة، الفعاليات، والدعم الفني.

## الملفات الرئيسية

- `style.css`: تعريف القالب داخل WordPress.
- `functions.php`: إعدادات القالب وتحميل ملفات CSS/JS.
- `front-page.php`: الصفحة الرئيسية.
- `template-parts/sections/`: أقسام الصفحة.
- `inc/theme-data.php`: بيانات الكليات وروابط وثائق الحوكمة.
- `assets/css/main.css`: التنسيق الكامل.
- `assets/js/main.js`: قائمة الجوال وتصفية شبكة الكليات.

## التركيب على WordPress

1. انسخ مجلد هذا القالب إلى:
   `wp-content/themes/sirte-elc`
2. من لوحة تحكم WordPress افتح:
   `المظهر > قوالب`
3. فعّل قالب:
   `Sirte E-Learning Center`
4. اجعل الصفحة الرئيسية تعرض الصفحة الأمامية من إعدادات القراءة إذا لزم الأمر.

## تعديل بيانات الكليات

افتح:
`inc/theme-data.php`

ثم عدل عناصر الدالة:
`sirte_elc_faculties()`

كل كلية لها:

```php
[
    'name' => 'اسم الكلية',
    'logo' => 'رابط الشعار',
    'url' => 'رابط تصنيف Moodle',
]
```

## تعديل وثائق الحوكمة

افتح:
`inc/theme-data.php`

ثم عدل عناصر الدالة:
`sirte_elc_documents()`

كل وثيقة لها:

```php
[
    'title' => 'عنوان الوثيقة',
    'description' => 'وصف مختصر',
    'url' => 'رابط Google Drive',
    'icon' => 'book',
]
```

الأيقونات المتاحة حالياً: `network`, `shield`, `target`, `book`.

## الملاحظات

- القالب يعتمد على خط Cairo من Google Fonts مع Tahoma كبديل.
- الصورة الرئيسية مأخوذة من منصة Moodle الحالية.
- شعار الجامعة/المركز يمكن استبداله لاحقاً من `المظهر > تخصيص > هوية الموقع`.
- لا توجد حاجة إلى إضافات خارجية لتشغيل الصفحة الرئيسية.
